### [CreditRiskScoring](https://github.com/kjd-dktech/CreditRiskScoring)

Predictive modelling for credit risk in the context of financial inclusion.

| | |
|---|---|
| **Problem** | Predict the probability of default for a loan applicant |
| **Approach** | Supervised classification with scikit-learn; SHAP for model interpretability; served via FastAPI backend with a Next.js frontend; full stack deployable with Docker Compose |
| **Stack** | `scikit-learn` `SHAP` `FastAPI` `Next.js` `Tailwind CSS` `Docker Compose` |

### [OCR_CNN_MNIST](https://github.com/kjd-dktech/OCR_CNN_MNIST)

Development and training of a Convolutional Neural Network for handwritten digit classification on the MNIST dataset.

| | |
|---|---|
| **Problem** | Handwritten digit recognition (OCR) |
| **Approach** | CNN designed, trained, and evaluated in Jupyter; model artefacts versioned; training metrics and visualisations exported to `Outputs/`; final model saved in Keras format |
| **Stack** | `TensorFlow` `Keras` `NumPy` `OpenCV` `scikit-learn` `Jupyter` |

[![Repo](https://img.shields.io/badge/GitHub-OCR__CNN__MNIST-181717?style=flat&logo=github&logoColor=white)](https://github.com/kjd-dktech/OCR_CNN_MNIST)

### [ewe-french_translator](https://github.com/kjd-dktech/ewe-french_translator)

Fine-tuning of M2M100 for French ↔ Ewé machine translation. Low-resource multilingual NLP pipeline.

| | |
|---|---|
| **Problem** | Machine translation for an under-resourced language (Ewé) |
| **Approach** | Fine-tuning a pre-trained multilingual sequence-to-sequence model (M2M100); full pipeline from preprocessing through BLEU evaluation to a Gradio demo |
| **Stack** | `PyTorch` `HuggingFace Transformers` `Gradio` |

[![Repo](https://img.shields.io/badge/GitHub-ewe--french__translator-181717?style=flat&logo=github&logoColor=white)](https://github.com/kjd-dktech/ewe-french_translator)
